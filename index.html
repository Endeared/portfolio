Weather Page
This website is an open-source project I made as part of the curriculum for The Odin Project's Full Stack course.

It operates using OpenWeatherMap's free API, and can query specific cities around the world and quickly return the current weather condition, as well as the day-by-day outlook for the next 7 days.
Built using HTML5, CSS3, JavaScript, and OpenWeatherMap API

Stock Ticker Page
This website is an open-source project I made during my own free time, as part of my interest in tracking Stocks / Foreign Exchange prices and values.

It utilizes Polygon's free API, and currently only supports Stocks listed on the NASDAQ or NYSE. In the future I plan to add support for Foreign Exchange pairs, as well as additional stocks.
Built using HTML5, CSS3, JavaScript, and Polygon API

Stock Ticker Bot (Discord)
This bot is part of the same open-source project that utilizes JS to pull stock prices / info from Polygon's API (described above).

In contrast, this bot uses Python and the discord.py library to parse through the API data and return the data as a cleanly-formatted embed message in a Discord channel.
Built using Python, discord.py, and Polygon API
